## Comandos iniciais

Na pasta do projeto:

```bash
git init
git add .
git commit -m "feat: release StockFlow S Plus stable version"
```

Depois crie o repositório no GitHub e conecte:

```bash
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/stockflow-s-plus.git
git push -u origin main
```
