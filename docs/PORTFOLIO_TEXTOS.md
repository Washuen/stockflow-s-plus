## Descrição para portfólio

StockFlow S Plus é um sistema empresarial fullstack desenvolvido para simular uma operação real de gestão de estoque, vendas, financeiro e usuários internos.

O projeto possui backend com Node.js e Express, banco PostgreSQL integrado via Prisma ORM, autenticação JWT, permissões por cargo, dashboard executivo, relatórios exportáveis em CSV, gestão de usuários com desativação segura e testes automatizados.

A interface foi construída com identidade visual SaaS corporativa, focada em clareza, análise de dados e experiência profissional.
