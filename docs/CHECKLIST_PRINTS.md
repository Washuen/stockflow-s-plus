# Checklist de Prints — StockFlow S Plus

Tire prints das seguintes telas para GitHub, portfólio e LinkedIn:

## Essenciais

- Tela de login.
- Dashboard executivo logado como Admin.
- Produtos.
- Estoque.
- Vendas.
- Financeiro.
- Relatórios profissionais.
- Exportação/geração de relatório.
- Usuários com permissões.
- Usuário desativado e botão Reativar.
- Terminal com testes passando.

## Prints técnicos

- Terminal com `npm run test`.
- Terminal com servidor rodando.
- Prisma/seed funcionando.
- Estrutura de pastas no VS Code.

## Ordem ideal no README

1. Login.
2. Dashboard.
3. Produtos.
4. Estoque.
5. Vendas.
6. Financeiro.
7. Relatórios.
8. Usuários/Admin.
9. Testes passando.
